# 要求内容

## 概要

Goバージョンを1.22から1.25にアップグレードする。

## 背景

プロジェクトで使用しているGoバージョン1.22から、最新の安定版である1.25にアップグレードすることで、以下のメリットを享受する:
- 新しい言語機能の利用
- パフォーマンス改善
- セキュリティパッチの適用

## 変更対象

1. **ドキュメント**
   - `docs/architecture.md`: テクノロジースタック表のGoバージョン記載

2. **ビルド設定**
   - `key-management-service/go.mod`: Goバージョン指定
   - `key-management-service/Dockerfile`: ベースイメージのGoバージョン

3. **ソースコード**
   - Go 1.25で追加・変更された機能を活用できる可能性を確認
   - 非推奨になった機能がないか確認

## 要求事項

- [ ] `docs/architecture.md`のGoバージョン表記を1.23.xから1.25.xに更新
- [ ] `key-management-service/go.mod`のgoディレクティブを1.25に更新
- [ ] `key-management-service/Dockerfile`のベースイメージを`golang:1.25-alpine`に更新
- [ ] 更新後、ビルドエラーがないことを確認
- [ ] 依存ライブラリとの互換性を確認

## 非機能要求

- 既存機能への影響がないこと
- テストが引き続き正常に動作すること
- ビルドプロセスに遅延が生じないこと
